# 가든 매치 - 텃밭 작물 추천 서비스

질문을 기반으로 한 맞춤형 텃밭 작물 추천 서비스의 소개 페이지입니다.

## 주요 기능

- 16가지 작물 유형 분석
- 맞춤형 작물 추천
- AI 기반 과학적 분석
- 반응형 웹 디자인

## 시작하기

### 필수 조건

- Node.js (버전 14 이상)
- npm 또는 yarn

### 설치

```bash
cd garden-recommend-intro
npm install
```

### 개발 서버 실행

```bash
npm start
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 빌드

```bash
npm run build
```

## 기술 스택

- React 18
- CSS3 (Grid, Flexbox)
- 반응형 디자인
- PWA 지원
